QR Code generator

Introduction

This app is a simple QR code generator built with React. Users can input text to generate a corresponding QR code and have the option to download the QR code as a PNG file with a custom filename.

Features

Generate QRCode:

Enter URL, links, or text to generate a QR Code instantly.

Save as PNG file:

Save the generated QR Code as PNG file. The file name is based on the text inputted by user.

User-Friendly interface:

Easy-to-use interface fr quick QR Code generation and saving.

Usage

To use the app:

  • Enter text into the input field to generate a QR code.
  • Optionally, enter a filename for the QR code image.
  • Click the "Save" button to download the QR code as a PNG file with the specified filename.
  • This simple, user-friendly QR code generator app can be useful for quickly creating and downloading QR codes.
